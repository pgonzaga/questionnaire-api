# QuestionnaireAPI

### Getting started

 ```bash
 git clone git@github.com:pgonzaga/questionnaire-api.git
 cd questionnaire-api
 npm install
 npm start
  ```
  
  ### Running the test suite and ESLint
  
   ```bash
 npm test
  ```
